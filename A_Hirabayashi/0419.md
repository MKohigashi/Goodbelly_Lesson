## 1. 字句構造
- Rubyのプログラムにおいて、`アルファベットの大文字、小文字は区別される`
- 識別子と一部のリテラルの途中を除いては 任意の場所に `空白文字やコメント` を置くことができる
  - 空白文字とは `スペース、タブ、垂直タブ、バックスペース、キャリッジリターン、ラインフィード、改ページ`のこと

```ruby
# アルファベットの大文字と小文字の区別
testVariable = 10
TestVariable = 20

puts testVariable # 出力: 10
puts TestVariable # 出力: 20

# 空白文字やコメントの使用例
test_variable = 30   # ここにコメントを書くこともできます
```

### 1-1. コメント
- 文字列中や文字リテラル `?#' 以外の #から行末までをコメントと見なす（無視する）
```ruby
# puts "出力されない"
puts "出力される"
```
### 1-2. 予約語
- 予約語は以下

```ruby
BEGIN    class    ensure   nil      self     when
END      def      false    not      super    while
alias    defined? for      or       then     yield
and      do       if       redo     true     __LINE__
begin    else     in       rescue   undef    __FILE__
break    elsif    module   retry    unless   __ENCODING__
case     end      next     return   until
```

## 2. プログラム構造
### 2-1. 式  
例:  
```ruby
puts true            # true を出力
puts (1 + 2) * 3     # 3 * 3 = 9 を出力
foo()                # foo メソッドを呼び出す
if test then ok else ng end   # test が真の場合は "ok" を、偽の場合は "ng" を出力
```

- Rubyのプログラムは式を並べたもの
- 式と式の間はセミコロン(;)または改行で区切る

```ruby
# 改行で区切る例
print "Hello, "
print "world!"

# セミコロンで区切る例
print "Hello, "; print "world!"
```

### 2-2. プログラムの終了
- Rubyは以下のものに出会うとプログラムを終了する
  - ファイルの終わり
  - `^D`(コントロール+D), `^z`（コントロール+Z）
  - `__END__` のみの行（前後に空白があると認識されない）

```ruby
# ファイルの終わりでプログラム終了
puts "プログラム実行中..."

# この行がファイルの終わりになるため、プログラムが終了します。
```
